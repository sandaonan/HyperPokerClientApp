# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1Nw7_vQc54qfDOs5UzuNO4sBJnkARYlq2

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy to Production (部署到线上)

想要在手机上访问你的网站？查看 [DEPLOY.md](DEPLOY.md) 获取详细的部署指南。

**快速部署到 Vercel：**
1. 将代码推送到 GitHub
2. 访问 [vercel.com](https://vercel.com) 并登录
3. 导入你的 GitHub 仓库
4. 点击部署，完成！

部署后你会得到一个网址（如 `https://your-app.vercel.app`），可以在手机上直接访问。
